#  Birthday Surprise Webpage with Rickroll Intro

This project is a fun birthday webpage with the classic Rickroll intro. When someone opens the page, blow the candle and gets rickroolled. This ensures that your friend has the best birthday greating ever.

---

## How to use

- **Download the files**  

- **Put your respective friends name in the code change the style and effects if you want**  

- **Put them in a zip file and send them (remember to hide the video)**
  
- **Say them to open the HTML file and watch them suffer**    

---

##  How It Works

1. The page loads and has a nice clean black minimalistic UI and a cake with candle.  
2. After the button click, the confetti works and the candle is blown and JavaScript switches to Rickroll.  
3. All styling and effects can be modified in the CSS/JS files.
